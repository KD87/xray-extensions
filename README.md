# X-Ray extensions

Проект представляет собой набор дополнений, расширяющих возможности игрового движка **X-Ray Engine** для скриптеров, а также, в ряде случаев, меняющих поведение игры. Изменения реализованы в виде бинарных патчей библиотек и исполняемых файлов.

## Сборка
Для сборки и получения файлов с правками необходимо:
1. Поместить оригинальные библиотеки с припиской **_orig** (например `xrGame_orig.dll`) в каталоги рядом с файлами **corrections_list.txt**.
2. Запустить командные файлы, которые лежат в корне проекта. Названия командных файлов соответствуют выбранной платформе и названию библиотек.

В результате создаются файлы, которые нужно переместить в каталог `bin` игры.

> Внимание! Оригинальные библиотеки не должны содержать каких-либо других правок.

## Описание
